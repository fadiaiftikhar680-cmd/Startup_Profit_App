# Startup Profit Prediction

## Description

This project predicts startup profit using Machine Learning and Linear Regression. Users can enter startup investment details and get an estimated profit prediction through a Streamlit web application.

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Joblib

## Project Files

- app.py
- model.pkl
- dataset.csv
- requirements.txt
- README.md

## How to Run

1. Open the project folder in VS Code.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Run the Streamlit application:

```bash
py -m streamlit run app.py
```

4. Open the URL shown in the terminal (usually http://localhost:8501).

## Features

- User-friendly web interface
- Real-time profit prediction
- Machine Learning based prediction model
- Streamlit deployment ready
